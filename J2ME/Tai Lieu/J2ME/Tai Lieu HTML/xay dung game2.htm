Khám phá Trò chơi API of MIDP 2.0

Tuesday, 29. May 2007, 13:04:52

J2ME Là một nền tảng đại chúng để phát triển những trò chơi những thiết bị vô tuyến. Với MIDP 2.0, một gói mới đã được giới thiệu mà cung cấp vài cơ cấu chơi mà chỉ đã khả dĩ print MIDP 1.0 với nhiều mã lặp lại. Trong phần này loạt dạy kèm này, Tôi sẽ cho bạn làm quen với gói chơi này và giúp đỡ bạn phát triển một trò chơi đơn giản mà sử dụng tất cả các lớp gói này như một công cụ học Gói được gọi là javax.microedition.lcdui.game,và nó những xây dựng ở trên những khái niệm mà bạn được học trong những sự xếp đặt trước đây trong loạt này.

J2ME Gaming API:
Có chỉ có năm lớp Print:javax.microedition.lcdui.game package: GameCanvas, Layer, Sprite, TiledLayer, and LayerManager.
Năm lớp này sẽ đủ cung cấp một nền tảng (cho) sự phát triển (của) những trò chơi với một phạm vi rộng (của) những khả năng.
The Layer class is the superclass of the Sprite and TiledLayer classes Lớp này trừu xuất hành vi (của) một phần tử trực quan trong một trò chơi.Phần tử này có thể Build Một sprite,mà đại diện cho một người độc lập đồ thị (hay một tập hợp (của) đồ họa (cho) hoạt cảnh) mà có thể được di chuyển xung quanh trò chơi screen...

Một Sách vỡ lòng rất Ngắn trên Tòa nhà Trò chơi:
Một trò chơi hay hoạt cảnh được xây dựng theo nguyên lý Của Lặp lại Thực hiện Một mẩu mã. Mẩu mã này theo dõi giá trị (của) những biến thể hiện và cập nhật trạng thái trò chơi tương ứng. Dựa vào trạng thái trò chơi, mã rồi những vẽ/ những sơn/ Sơn lại màn ảnh trò chơi với những phần tử mà tạo ra trò chơi. Những giá trị (của) những biến thể hiện có thể thay đổi bởi vì hoặc sự tương tác người sử dụng hoặc hành vi trò chơi bên trong.

// lớp chính
public MainClass {
private GameCanvas canvas = new MyGameCanvas();

}

// phần còn lại (của) mã
}


// lớp bản vẽ thực tế
public MyGameCanvas extends GameCanvas implements Runnable {


public MyGameCanvas() {
// sự khởi tạo mã
}

public void start() {
// làm sự khởi tạo

// và sau đó bắt đầu một luồng
Thread runner = new Thread(this);
runner.start();
}

private void run() {

// hay trong khi (keeprun = thật)
//ở đâu keeprunning là một biến thể hiện

while(true) {


// những sự kiểm tra nếu trò chơi được đạt đến
// một số trạng thái ranh giới hay điều kiện bắt buộc
verifyGameState();


// đưa đầu vào từ người sử dụng và
// cập nhật những biến thể hiện
// mà mô tả những phần tử những trò chơi

checkUserInput();


// những phần tử những sơn trên màn ảnh để phản chiếu
// trạng thái trò chơi hiện thời sử dụng dòng
// đối tượng graphics

updateGameScreen(getGraphics());


// kiểm soát nhịp độ mà những sự cập nhật màn ảnh được làm

Thread.sleep(milliseconds);
}
}
}
